## Welcome to Blog App
  
This is a **modern blogging platform** built with Next.js, featuring user authentication, blog post management, and a clean, intuitive interface.

### Features

* **Create and Manage Blogs** - Write and publish your thoughts
* **User Profile** - Personalized user pages
* **Authentication** - Secure login and registration

### Getting Started

To start exploring:

1. Register for a new account
2. Login to access your dashboard
3. Create a new blog post
4. View all blogs and users

<hr/>
Built using Next.js, React, and Drizzle ORM (PostgreSQL)

Source code [https://github.com/johnpaulbelaos/fullstackopen-nextjs](https://github.com/johnpaulbelaos/fullstackopen-nextjs)
